
# Mamta Survase | Portfolio Website

## Overview
This is an interactive portfolio website built using **HTML, CSS, and JavaScript**. It shows my skills, projects, and contact information in a modern, responsive layout.

## Features
- **About Me:** Profile, introduction, and resume download.  
- **Skills:** Animated skill bars with descriptions.  
- **Contact:** Form with validation and contact info card.  
- **Dark Mode:** Switch between light and dark themes.  
- **To-Do List:** Add, remove, and complete tasks.  
- **Typing Animation:** Header text animates like typing.

## Setup
1. Download or clone the project.  
2. Keep the folder structure like this:
```

project-folder/
├─ index.html
├─ style.css
├─ script.js
├─ images/profile.jpg,project1.jpg,project2.jpg.project3.jpg
└─ resume.pdf
├screenshots/
• HomePage,
• AboutSection,
• SkillsSection,
• ImageGallery,
• ToDoList,
• ContactSection
• DarkMode

```
3. Open `index.html` in a browser.  

## Code Structure
- **index.html** – Website structure.  
- **style.css** – Design and layout.  
- **script.js** – Interactivity (form validation, dark mode, skill animation, to-do list).
